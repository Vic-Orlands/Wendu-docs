## Getting Started

Wendu is built with [Next.js](https://nextjs.org/), [TypeScript](https://www.typescriptlang.org/), [Tailwind CSS](https://tailwindcss.com/), and [shadcn/ui](https://ui.shadcn.com/) components. This documentation will guide you through the development structure so you can easily extend its features and modify the codebase.
